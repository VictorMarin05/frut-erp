# Frut-ERP

Sistema de gestión de logística agrícola, inventario y reparto.

Este proyecto personal nace de mi interés por unir la arquitectura de software con las necesidades reales de gestión empresarial, aplicando los conocimientos de mi doble grado en Ingeniería Informática y ADE.

## Estrategia de Desarrollo

Para la construcción de este ERP he optado por un enfoque dual:

* **Frontend:** La interfaz y la experiencia de usuario están siendo maquetadas y estructuradas con la asistencia de Inteligencia Artificial. Esto me permite iterar rápidamente sobre el diseño (React y Tailwind v4) sin perder foco en la lógica de negocio.
* **Backend:** El núcleo del sistema, la base de datos y la infraestructura están desarrollados 100% a mano desde cero. El objetivo es profundizar en la arquitectura de software, el modelado relacional y la inyección de dependencias.

## Stack Tecnológico

* **Backend:** Python, FastAPI, SQLModel.
* **Base de datos:** PostgreSQL.
* **Infraestructura:** Docker y Docker Compose.
* **Frontend:** React, Vite, Tailwind CSS v4.
